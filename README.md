# Electron typescript boilerplate

## Setup

`npm install -g typescript`

`npm install`

`npm run-script post-npm-install`

## Build executables

- All executables :
`npm run-script package`
- OSX :
`npm run-script package-darwin`
- Linux :
`npm run-script package-linux`
- Windows :
`npm run-script package-windows`
